================================================================
PHYSICS IMPLEMENTATION DETAILS
================================================================

Coordinate System: Boyer-Lindquist (t, r, θ, φ)
Integration Method: Runge-Kutta 4th order (RK4)
Precision: Single precision (float) with renormalization
Step Size: Adaptive (0.02-0.10 based on curvature)
Max Steps: 512 per ray
Ray Direction: Backward from camera to sources
Termination: Horizon, escape, disk hit, max steps

Kerr Metric Components:
-----------------------
g_tt = -(1 - 2Mr/Σ)
g_tφ = -2Mar sin²θ/Σ
g_rr = Σ/Δ
g_θθ = Σ
g_φφ = [(r²+a²)² - a²Δsin²θ]sin²θ/Σ

Where:
  Σ = r² + a²cos²θ
  Δ = r² - 2Mr + a²

Geodesic Equation:
------------------
d²x^μ/dλ² = -Γ^μ_νρ (dx^ν/dλ)(dx^ρ/dλ)

Computed using explicit Christoffel symbols derived from metric.

Accretion Disk Model:
---------------------
Inner radius: 3M (ISCO approximation)
Outer radius: 12M
Thickness: ±0.15 radians
Temperature: T ∝ r^(-3/4)
Emission: I ∝ r^(-3)
Rotation: Keplerian (Ω = √(M/r³))

Redshift Factor:
----------------
g = (k_μ u^μ_obs) / (k_μ u^μ_emit)

Intensity boost: I_obs = g³ · I_emit

SCIENTIFIC ACCURACY
================================================================

This implementation is based on peer-reviewed physics:

Kerr Metric:
- Kerr (1963): Original derivation
- Boyer & Lindquist (1967): Coordinate system

Geodesic Integration:
- Chandrasekhar (1983): Mathematical framework
- Levin & Perez-Giz (2008): Numerical methods

Black Hole Imaging:
- Luminet (1979): First computer simulation
- James et al. (2015): Interstellar methods
- EHT Collaboration (2019): M87 observations

Accretion Disk:
- Shakura & Sunyaev (1973): Thin disk model
- Novikov & Thorne (1973): Relativistic disks

Known Approximations:
---------------------
✓ Thin disk approximation (no vertical structure)
✓ Simplified emissivity (I ∝ r^(-3))
✓ No radiative transfer (optically thin)
✓ Single-scattering only (no multiple lensing)
✓ Geometric optics (ray-tracing, not wave optics)
✓ Test particle approximation (photons don't affect spacetime)

These are standard simplifications for real-time visualization.
